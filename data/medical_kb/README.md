# HealthBot Medical KB

This corpus is intentionally small and broad enough for a general medical education bot.

## Purpose

- cover common adult primary-care questions
- support grounded answers with source metadata
- keep the corpus small enough to inspect manually
- include some urgent-care patterns for safer routing and review

## Selection criteria

- patient-facing, general-audience health education pages
- official public-health or government medical sources
- spread across common question types: respiratory, cardiometabolic, GI, urinary, pain, neurology
- short hand-written summaries rather than copied page text

## Included topics

- `asthma.md`
- `hypertension.md`
- `type_2_diabetes.md`
- `flu_symptoms.md`
- `flu_treatment.md`
- `viral_gastroenteritis.md`
- `adult_bladder_infection_uti.md`
- `back_pain.md`
- `migraine.md`
- `urgent_red_flags.md`

## Notes

- Each file includes source name, source URL, topic, retrieval date, and a usage note.
- The summaries are educational only and should not be treated as diagnosis or treatment instructions.
- This is a starter corpus. It is broad enough for a demo, not broad enough for open-ended clinical coverage.
